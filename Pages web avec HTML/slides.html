Un peu d'histoire

Naissance

Depuis la fin des années 1960, on développe les premiers langages balisés ayant pour but de séparer la forme et le fond des documents (GML 1969-78, SGML 1986).

Ces formats ont été utilisés pour créer HTML en 1993, qui est une version simplifiée de SGML. Elle contient entre autres des hyperliens, qui permettent de relier les documents sur internet.

D'autres formats sont également inspirés de cette idée (XML 1998, XHTML 2000).

En clair

HTML est un format en texte brut qui permet de créer des documents, potentiellement liés les uns aux autres. Ces documents peuvent être mis sur internet pour constituer des sites internet. Ils sont généralement affichés par un navigateur.

Fonctionnement global

Langage balisé

HTML est un langage balisé, c'est à dire qu'il permet de créer une hiérarchie de contenus en utilisant une grammaire spécifique.

Exemple


<html>
  <head>
    <title>Document HTML</title>
  </head>
  <body>
    <h1 id="title">Titre</h1>
    <p class="super">paragraphe</p>
  </body>
</html>

Il est possible de voir le HTML de tous les sites à partir d'un navigateur.

Balises courantes

Titres

6 niveaux de titres sont disponibles, du plus important h1 au moins important h6.

Texte

On peut créer des paragraphes p, et des listes ul comprenant des éléments li.

Structure de page

La balise html est la balise racine du document.

De nombreuses balises servent à englober du contenu : body, main, header, footer, article…

Métadonnées

Les métadonnées sont des données qui ne font pas partie du contenu, mais qui permettent de décrire et d'agrémenter le document. Elles sont indiquées dans la balise head.

Parmi les métadonnées généralement utilisées, on note le titre, l'auteur, les mots-clés, mais également les feuilles de style (CSS) et les scripts (JavaScript).
